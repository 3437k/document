### Setting

```bash
git config --global user.email "<email@naver.com>"

git config --global user.name "<user name>"

git config --global --list
```

### Init 
```bash
git init
```

### Staged
```bash
git add .
git add README.txt
```
### Commit
```bash
git commit -m "message"
```

### Push
```bash
git remote add origin https://github.com/<ID>/<REPO>.git

git push origin master
```

### Checkout 
```bash
git log 

git checkout <SHA>

git checkout -  
```
